## dataos-ctl operate pulsar resource-quotas reset

Reset the specified namespace bundle's resource quota to default value.

### Synopsis

USED FOR:
    Reset the specified namespace bundle's resource quota to default value.

REQUIRED PERMISSION:
    This command requires super-user permissions.

OUTPUT:
    #normal output
    Reset resource quota successful



```
dataos-ctl operate pulsar resource-quotas reset [flags]
```

### Examples

```
    #Reset the specified namespace bundle's resource quota to default value
    pulsarctl resource-quotas reset (namespace name) (bundle range)


```
