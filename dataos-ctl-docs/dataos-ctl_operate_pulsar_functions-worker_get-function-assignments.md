## dataos-ctl operate pulsar functions-worker get-function-assignments

Get the assignments of the functions across the worker cluster

### Synopsis

USED FOR:
    Get the assignments of the functions across the worker cluster

REQUIRED PERMISSION:
    This command requires super-user permissions.

OUTPUT:
    #normal output
    [ ]



```
dataos-ctl operate pulsar functions-worker get-function-assignments [flags]
```

### Examples

```
    #Get the assignments of the functions across the worker cluster
    pulsarctl functions-worker get-function-assignments


```
