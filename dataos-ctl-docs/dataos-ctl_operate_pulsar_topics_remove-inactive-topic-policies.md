## dataos-ctl operate pulsar topics remove-inactive-topic-policies

Remove inactive topic policies from a topic

### Synopsis

USED FOR:
    Remove inactive topic policies from a topic

REQUIRED PERMISSION:
    This command requires tenant admin permissions.

OUTPUT:


```
dataos-ctl operate pulsar topics remove-inactive-topic-policies [flags]
```

### Examples

```
    #Remove inactive topic policies from a topic
    pulsarctl topics remove-inactive-topic-policies [topic]


```
